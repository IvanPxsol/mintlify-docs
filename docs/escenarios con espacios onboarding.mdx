---
title: "Gestión de Escenarios de Tarifas"
description: "Aprende a crear y aplicar conjuntos de precios predefinidos (escenarios) para actualizar masivamente las tarifas por temporadas o eventos."
---

Esta funcionalidad te permite crear conjuntos de precios predefinidos, llamados **escenarios**, para distintas categorías de habitaciones. Puedes aplicar estos escenarios a planes tarifarios específicos en uno o varios rangos de fechas, lo que facilita la actualización masiva de tarifas para temporadas altas, eventos especiales u otras ocasiones.

## Creación de un Escenario

Sigue estos pasos para crear un nuevo escenario de tarifas.

<Steps>
  <Step title="Navega a la sección de Escenarios">
    Para comenzar, ve a la pestaña **Escenarios**. Aquí verás una lista de los escenarios que ya existen.
    <Frame caption="Pestaña de Escenarios mostrando la lista de configuraciones existentes.">
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2025/08/20/screenshot_1755721270_0_68a62e36008cd.jpg" alt="Vista de la pestaña de Escenarios con una lista de escenarios existentes." />
    </Frame>
  </Step>

  <Step title="Inicia un nuevo escenario">
    Haz clic en el botón para crear un nuevo escenario.
    <Frame caption="Botón para iniciar la creación de un nuevo escenario de tarifas.">
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2025/08/20/screenshot_1755721270_1_68a62e361bf0c.jpg" alt="Interfaz para la creación de un nuevo escenario." />
    </Frame>
  </Step>

  <Step title="Asigna un nombre">
    Ingresa un nombre claro y descriptivo en el campo **Nombre** para identificar fácilmente el escenario.
    <Frame caption="Campo 'Nombre' para identificar el escenario.">
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2025/08/20/screenshot_1755721270_2_68a62e362a68a.jpg" alt="Ingresando un nombre en el formulario del nuevo escenario." />
    </Frame>
  </Step>

  <Step title="Selecciona la moneda">
    Elige la moneda que se aplicará a todos los precios dentro de este escenario desde el menú desplegable.
    <Frame caption="Menú desplegable para seleccionar la moneda del escenario.">
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2025/08/20/screenshot_1755721270_3_68a62e363eca8.jpg" alt="Seleccionando una moneda desde la lista desplegable." />
    </Frame>
  </Step>

  <Step title="Define los precios">
    Establece el precio para cada categoría de habitación en los campos correspondientes.
    <Frame caption="Campos para ingresar los precios por cada tipo de habitación.">
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2025/08/20/screenshot_1755721270_4_68a62e365ba0a.jpg" alt="Estableciendo los precios para diferentes categorías de habitaciones." />
    </Frame>
  </Step>

  <Step title="Guarda el escenario">
    Confirma la creación haciendo clic en el botón para guardar los cambios.
    <Frame caption="Botón para guardar y confirmar la creación del escenario.">
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2025/08/20/screenshot_1755721270_5_68a62e366e47b.jpg" alt="Haciendo clic en el botón para guardar el escenario." />
    </Frame>
    <Check>
      ¡Escenario creado! Ahora aparecerá en tu lista de escenarios disponibles.
    </Check>
  </Step>
</Steps>

## Aplicación de un Escenario

Una vez que has creado un escenario, puedes aplicarlo a un plan tarifario para actualizar sus precios en fechas específicas.

<Steps>
  <Step title="Inicia la aplicación">
    Haz clic en el botón **+ Aplicar escenario** para abrir el formulario de aplicación.
    <Frame caption="Botón para comenzar el proceso de aplicación de un escenario.">
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2025/08/20/screenshot_1755721270_6_68a62e36813d3.jpg" alt="Botón para aplicar un escenario a un plan tarifario." />
    </Frame>
  </Step>

  <Step title="Selecciona el plan tarifario">
    En el menú desplegable **Tarifario**, elige el plan al que deseas aplicar los precios del escenario.
    <Note>
    En esta lista únicamente se muestran los planes tarifarios independientes (maestros).
    </Note>
    <Frame caption="Selección del plan tarifario desde un menú desplegable.">
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2025/08/20/screenshot_1755721270_7_68a62e36af081.jpg" alt="Elegiendo un plan tarifario del menú." />
    </Frame>
  </Step>

  <Step title="Elige el escenario">
    Selecciona el escenario que deseas aplicar desde el menú desplegable **Escenario**.
    <Info>
    La lista se filtra automáticamente para mostrar solo los escenarios que coinciden con la moneda del plan tarifario seleccionado.
    </Info>
    <Frame caption="Selección del escenario de tarifas a aplicar.">
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2025/08/20/screenshot_1755721270_8_68a62e36be92e.jpg" alt="Elegiendo un escenario desde el menú." />
    </Frame>
  </Step>

  <Step title="Define el rango de fechas">
    Usa el calendario en **Rango de fechas** para seleccionar las fechas de inicio y fin del período en que se aplicará el escenario.
    <Frame caption="Selector de calendario para definir el período de aplicación.">
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2025/08/20/screenshot_1755721270_9_68a62e36d13e5.jpg" alt="Usando el calendario para seleccionar un rango de fechas." />
    </Frame>
  </Step>

  <Step title="Agrega períodos adicionales (opcional)">
    Si necesitas aplicar el escenario a otros períodos no consecutivos, haz clic en **+ Agregar fecha** y selecciona los rangos adicionales.
    <Frame caption="Opción para añadir múltiples rangos de fecha no consecutivos.">
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2025/08/20/screenshot_1755721270_10_68a62e36e3523.jpg" alt="Agregando un segundo rango de fechas a la aplicación." />
    </Frame>
  </Step>

  <Step title="Revisa los cambios">
    Haz clic en **Revisar cambios** para ver un resumen de la operación. En la pantalla de revisión, verifica que los rangos de fechas y los nuevos valores de tarifas sean correctos.
    <Frame caption="Pantalla de revisión que resume los cambios antes de aplicarlos.">
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2025/08/20/screenshot_1755721271_12_68a62e372cdb5.jpg" alt="Resumen de los cambios de tarifas y fechas para verificación final." />
    </Frame>
  </Step>

  <Step title="Confirma y aplica">
    Para finalizar el proceso, haz clic en **Aplicar cambios**. El sistema actualizará las tarifas del plan seleccionado para las fechas especificadas.
    <Frame caption="Botón final para confirmar y aplicar los cambios de tarifas.">
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2025/08/20/screenshot_1755721271_13_68a62e374552f.jpg" alt="Haciendo clic en el botón 'Aplicar cambios' para finalizar." />
    </Frame>
    <Check>
      Los precios del plan tarifario se han actualizado correctamente.
    </Check>
  </Step>
</Steps>

## Verificación de Escenarios Aplicados

Puedes consultar un historial de todos los escenarios que se han aplicado para mantener un control de los cambios.

<Steps>
  <Step title="Accede al historial de aplicaciones">
    Ve a la pestaña **Escenarios aplicados** para ver el registro.
    <Frame caption="Pestaña 'Escenarios aplicados' para ver el historial.">
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2025/08/20/screenshot_1755721271_14_68a62e375f119.jpg" alt="Navegando a la pestaña de escenarios aplicados." />
    </Frame>
  </Step>
  
  <Step title="Verifica la aplicación">
    Consulta la lista para confirmar que los escenarios se aplicaron correctamente en los rangos de fecha definidos.
    <Check>
    El sistema mostrará un registro detallado de cada rango de fechas con su respectivo escenario y plan tarifario activo, confirmando que los cambios se aplicaron correctamente.
    </Check>
    <Frame caption="Lista detallada de los escenarios aplicados con sus rangos de fecha y planes tarifarios.">
      <img src="https://files-public-web.s3-us-west-2.amazonaws.com/1978/video_screenshots/2025/08/20/screenshot_1755721271_15_68a62e37779f5.jpg" alt="Historial de aplicaciones de escenarios, mostrando fechas y planes afectados." />
    </Frame>
  </Step>
</Steps>
